# Wanderlust 🌍

A full-stack travel listing web app inspired by Airbnb.

## 🚀 Features
- Search listings by location, country, title
- Category-based filtering
- User authentication (login/signup)
- Add/Edit/Delete listings
- Review system

## 🛠 Tech Stack
- Node.js
- Express.js
- MongoDB
- EJS
- Bootstrap
